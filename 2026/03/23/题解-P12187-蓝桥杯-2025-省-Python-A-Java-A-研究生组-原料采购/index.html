思路

可以想到按位置顺序来遍历所有采购点。每次到一个位置 后，答案就是 。这里的 指的是所有货本身的价钱。

现在问题就变成了如何快速求出位置 及之前的采购点中所有采购方式中货价最少的一种。

考虑反悔贪心。每次到一个位置，如果总数还不够 ，就无脑填上。否则，把它和之前所有单价进行比较，找到一个单价最大的。

  • 如果之前的最大单价大于当前的单价，就反悔掉当前 的那一部分，然后维护答案。
  • 如果之前的最大单价小于当前单价，那么当前单价对答案一定没有贡献，直接扔掉。
